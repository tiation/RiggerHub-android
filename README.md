## 🚀 Technology Stack

- **Language**: Kotlin 1.9.20
- **Architecture**: MVVM
- **UI Framework**: Jetpack Compose
- **Minimum Android Version**: API 24 (Android 7.0)
- **Target Android Version**: API 34 (Android 14)
